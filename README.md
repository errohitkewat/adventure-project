# Adventure Project

A modern adventure booking website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

This project is designed for showcasing adventure activities in Manali with a premium UI. It includes activity sections, pricing, gallery, testimonials, booking form, and interactive popups for a better user experience.

---

## Features

- Modern and responsive UI
- Adventure activity cards with detailed popups
- Booking form with pricing summary
- Add-ons like photo, video, and GoPro recording
- Gallery section
- Testimonials section with review popup
- Smooth premium design with dark theme
- Built using reusable components

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React
- React Icons
